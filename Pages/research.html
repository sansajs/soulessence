Comparing the reference website with my clothing website

Yala Design is a women clothing store which sells variety of comfortable clothes for women of any ages. I have taken reference from this website because this website has a very minimalistic theme which I had also wanted to implement on mine. So, with the help of this website, I have taken ideas for my homepage, where I have designed the theme in a very similar manner. I have kept a banner at the most top of the website which shows the order shipping offer, then I have kept some search button and my cart icons imported from online.



Zaful is an international brand of women’s clothing store. This website makes various offers every week and every day in order to attract more clients. Here, from above we can see that there is a “Hand-picked” clothing section in their website with various sale items displayed. So, in order to make my website look more practical and attractive, I have kept a hand-picked clothes section in my website with discounts offer displayed.



Every website should contain a contact section where the buyer can inquire and give feedbacks about the products or services to the company. From the website clothingshoponline, I have taken idea about the format of the contact section should be, so that I can implement on my website too. I have kept a clear information about the store’s location where I have imported a map from the google maps and kept in the contact section.



For the product description section, I have referred to a Nepali clothing website named DS Collection. With the help of this website, I learned about how are product described and kept them for sale. So, using this idea I have kept product description of each of the products and have rated them and given a short description about the clothes and shoes and the material they are made of. I have also kept a review section, where the buyers can give reviews about the product and send it to the organization. After, viewing about the product description, the buyer then can purchase the item by filling the options such as size, color and the quantity they want to buy.



Dotti is an Australian brand company which sells women’s clothes and accessories. This website has displayed its products is a very simple and minimalistic way that makes the buyers easy to allocate the products and buy them. Using this idea, I have tried to use similar pictures and used the same pattern to display clothes and accessories in my website.
